# UVM-SFSoC-v1 项目说明（方案1：低功耗传感器融合 SoC）

## 1. 项目目标
构建一个面向可穿戴、工业监测、智能家居场景的小型 SoC 前端开发与验证项目，核心能力为：

- 多路 I2C 传感器采集
- 低功耗自治搬运（uDMA-like）
- 轻量数据融合与事件上报
- 完整 UVM 验证闭环（功能、异常、低功耗场景）

---

## 2. 目录规划
根目录按以下结构组织：

- `rtl/`：RTL 设计代码
- `sim/`：仿真目录
  - `sim/uvm/`：UVM 验证平台
  - `sim/work/`：编译与运行工作目录
- `syn/`：综合（DC）脚本与结果
- `fm/`：形式等价检查（Formality）脚本与结果
- `md/`：项目文档（本文件及后续规格文档）

## 3. SoC 顶层定义（v1）

### 3.1 核心与总线
- CPU：RV32IMC 单核 MCU
- 总线：AHB-Lite + APB 分层结构
- 存储：Boot ROM + SRAM（含 retention bank 规划）

### 3.2 外设
- I2C Master x2（100k/400k/1MHz，7-bit 地址，repeated-start）
- SPI/QSPI Master x1
- UART x1
- GPIO / Timer / WDT / RTC

各外设在本项目中的具体作用：

- **I2C0 / I2C1（双路传感采集主通道）**
  - I2C0 主要连接 IMU/加速度计等高频传感器，I2C1 主要连接温湿度/环境类传感器。
  - 支持 repeated-start 以减少总线释放开销，适合“寄存器地址写入 + 连续读取”场景。
  - 采集完成事件可触发中断，并可作为低功耗唤醒源之一。

- **SPI/QSPI（高速外设与扩展存储通道）**
  - 用于连接高速外设或外部 Flash，承担比 I2C 更高吞吐的数据交换。
  - 在本项目中可作为高频数据源补充通道，或用于固件/参数扩展存储访问。

- **UART（调试与事件上报通道）**
  - 用于输出融合结果、告警事件、运行状态日志。
  - 在 bring-up 与回归阶段承担可观测性接口，便于快速定位问题。

- **GPIO（外部事件输入与控制输出）**
  - 作为按键/中断脚/传感器 ready 信号输入，支持边沿触发唤醒。
  - 也可作为简单控制输出（如使能脚、告警灯）。

- **Timer（周期调度与超时基准）**
  - 提供系统周期任务节拍（例如轮询窗口、软定时任务）。
  - 用于驱动采样周期管理与部分协议超时控制。

- **WDT（系统健壮性保障）**
  - 监控系统是否卡死，超时触发复位或告警中断。
  - 在低功耗与异常恢复场景中提供最后防线。

- **RTC（低功耗时基与定时唤醒核心）**
  - 在 AON 域持续运行，提供长周期低功耗计时。
  - 按计划周期唤醒系统执行“采集 -> 搬运 -> 处理 -> 上报”流程。

- **uDMA（外设到SRAM自治搬运引擎）**
  - 将 I2C/SPI/UART 数据直接搬运到 SRAM，降低 CPU 搬运开销。
  - 是“低功耗高采样效率”的关键模块，支持完成中断通知上层处理。

- **Event INTC（中断汇聚与分发）**
  - 汇聚 RTC/GPIO/I2C/UDMA/UART/SPI/FUSION 等事件源。
  - 提供屏蔽、挂起、清除等能力，统一向 CPU 输出中断入口。

- **Fusion（轻量融合与事件检测）**
  - 对采样数据执行滑动平均与阈值判断。
  - 命中阈值后上报事件，触发中断并驱动 UART 上报路径。

### 3.3 数据通路
- uDMA-like：支持 I2C/SPI/UART 数据搬运至 SRAM
- Ring Buffer：采样数据缓存
- 轻量前处理：滑动平均 + 阈值触发（事件模式）

### 3.4 低功耗
- 电源域：AON + SYS
- 工作模式：RUN / SLEEP / DEEP-SLEEP
- 唤醒源：RTC / GPIO / I2C 传输完成 / WDT

---

## 4. 典型业务流程
1. RTC 定时唤醒 SoC
2. I2C0 读取 IMU，I2C1 读取温湿度
3. uDMA 将数据写入 SRAM 缓冲
4. CPU 执行轻量融合与阈值判断
5. 命中事件则 UART 上报，否则回到低功耗模式

---

## 5. 验证目标（UVM）

### 5.1 基础功能
- I2C 读写正确性（单次/连续/repeated-start）
- uDMA 搬运正确性（地址、长度、中断）
- APB 寄存器可编程性与复位值检查

### 5.2 异常与鲁棒性
- I2C NACK、时钟拉伸超时、总线忙冲突
- DMA 边界条件（0 长度、非对齐、跨边界）
- 中断屏蔽/清除/并发触发

### 5.3 低功耗验证
- RUN/SLEEP/DEEP-SLEEP 切换
- 唤醒源有效性及时序正确性
- 低功耗模式下外设自治行为约束
